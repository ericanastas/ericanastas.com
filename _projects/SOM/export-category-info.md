Exports an Excel file of the values below for all the categories in the active document.

- Name
- Id
- BuiltInCategory
- ParentCategory
- CategoryType
- IsCuttable
- IsTagCategory
- CanAddSubcategory
- AllowsBoundParameters
- HasMaterialQuantities
- LineColor
- Material
- CutLineWeight
- ProjectionLineWeight
